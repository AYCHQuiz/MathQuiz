Emathquiz

Create mathematical quizes

EmathQuiz is a tool for creating different kind of quizes. It depends on jQuery, JSXGraph and MathQuill. Optionally mqpanel.js can also be used.

One questionset or -type is a function that returns a randomized question object. Question object includes the question text, possible image (or data for drawing that image), feedbacks for correct and wrong answers and a function that checks the users answer. Questionsets are given to EmathQuiz as an array of these functions. EmathQuiz gets question after another from these sets by selecting functions randomly from this array and executing them.

Possible question types include: multichoise questions, short answer questions and questions that are answered by manipulating the given image. There are some example questionsets built in jquery.emathquiz.js.

Here are couple of examples, how to use emathquiz.js.

Example 1: Interactive images

Quiz questions can have interactive images that the user can manipulate. The correctness of the answer is determined from the image.

<div id="box1"> </div> <script type="text/javascript"> jQuery('#box1').emathquiz({ title: "Angles of triangles", overlay: false, func: [Quizes.triangletype_image] }); </script>

Example 2: Support for localization

The system support localization of quizes. The language is determined by finding the closest parent in DOM-tree, that has lang-attribute set.

Built-in quizes support English, Finnish and Swedish. Adding languages is easy.

<div id="box2" lang="fi"> </div> <script type="text/javascript"> jQuery('#box2').emathquiz({ title: "Kulmien tyypit", overlay: false, func: [ Quizes.angletype_image ] }); </script>

Example 3: Division of a segment

Image and question text can have different layout. The feedback can give a detailed correct answer.

<div id="box3" lang="sv"> </div> <script type="text/javascript"> jQuery('#box3').emathquiz({ title: "sträckan", overlay: false, func: [Quizes.segment_image] }); </script>

Example 4: Multichoise answers

EmathQuiz supports also other answertypes like multichoise and short answer.

<div id="box4"> </div> <script type="text/javascript"> jQuery('#box4').emathquiz({ title: "Angles of triangles", overlay: false, func: [ Quizes.triangle_multi ] }); </script>

Example 5: Shortanswer and multiple questionsets

Multiple questionsets can be combined in the same quiz.

Short answer type uses MathQuill for input.

<div id="box5"> </div> <script type="text/javascript"> jQuery('#box5').emathquiz({ title: "Angles of triangles", overlay: false, func: [ Quizes.triangle_multi, Quizes.triangle_short ] }); </script>
